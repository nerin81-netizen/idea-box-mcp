## ✨ ये क्या है?

**तुम्हारे पास Markdown फाइल्स से भरे फोल्डर्स हैं।** आइडियाज जो तुमने कलेक्ट किए हैं। मेमोस, लिंक्स, रेफरेंसेस। तुम्हें पता है वो वहां हैं — पर जब जरूरत हो तो कैसे ढूंढोगे? यही प्रॉब्लम है।

`idea-box-mcp` तुम्हारे लोकल Markdown फोल्डर्स को **LLM-callable टूल्स** में बदल देता है। इसके बजाय:

```bash
# ❌ पहले: मैनुअल सर्चिंग
grep -r "AI" ~/ideas-h/
find ~/memos-h/ -name "*.md" -exec grep -l "arxiv" {} \;
```

अब तुम बस बोल सकते हो:

```text
# ✅ अब: नैचुरल लैंग्वेज
"AI के आइडियाज ढूंढो"
"arxiv वाले मेमोस दिखाओ"
"ये सेव करो — ये लिंक बाद में पढूंगा"
```

**दो एक्सपोज्ड डायरेक्टरीज:**

| डायरेक्टरी | अंदर क्या है | कब यूज़ करोगे |
|---|---|---|
| `~/ideas-h/` | ऐप्स के आइडियाज (स्टेटस/ग्रेड के साथ) | "अब क्या बनाऊं?" |
| `~/memos-h/` | रॉ रेफरेंसेस (8 कैटेगरीज) | "पहले क्या सेव किया था?" |

> **कोर आइडिया:** तुम्हारे लोकल Markdown फोल्डर्स पहले से ही एक नॉलेज बेस हैं। ये MCP बस उन्हें LLMs के लिए *क्वेरीएबल* बना देता है।

## 🛠 टूल्स (5)

| टूल | आर्ग्स | डिस्क्रिप्शन | एग्जांपल |
|---|---|---|---|
| `search_ideas` | `keyword`, `limit` | कीवर्ड से आइडियाज सर्च करो | "AI आइडियाज ढूंढो" |
| `count_ideas` | — | स्टैटिस्टिक्स (टोटल, ग्रेड वाइज़, रीसेंट) | "मेरे आइडियाज कितने हैं?" |
| `search_memos` | `keyword`, `category`, `limit` | मेमोस सर्च करो (8 कैटेगरीज) | "arxiv मेमोस ढूंढो" |
| `count_memos` | — | स्टैटिस्टिक्स (टोटल, कैटेगरी वाइज़) | "मेमोबॉक्स के स्टैट्स दिखाओ" |
| `add_memo` | `title`, `content`, `category`, `tags` | ऑटो-कैटेगराइज़ेशन के साथ नया मेमो ऐड करो | "ये सेव करो — ..." |

### रियल-वर्ल्ड सीनारिओज़

**सीनारियो 1: "वो हेल्थ ट्रैकिंग वाला आइडिया क्या था?"**

तुम्हें याद है 2 हफ्ते पहले हेल्थ ऐप के बारे में कुछ लिखा था, पर मिल नहीं रहा। बस पूछो:

```text
"हेल्थ के आइडियाज ढूंढो"

→ 🔍 'हेल्थ' के रिजल्ट्स — 145 में से 3

  · **इंटीग्रेटेड हेल्थ डेटा ऐप** [B]
    हॉस्पिटल्स, फार्मेसीज, इंश्योरेंस का डेटा एक जगह...
    📁 `2026-06-15_integrated-health-data-app.md`

  · **मेडिकेशन अलर्ट + इंटरैक्शन्स**
    गोलियां लेते समय साइड इफेक्ट्स चेक करने वाला ऐप...
    📁 `2026-06-20_medication-alert.md`
```

**सीनारियो 2: "ये बाद में पढूंगा"**

तुम ब्राउज़ कर रहे हो और कुछ इंटरेस्टिंग मिला। बुकमार्क करने के बजाय (और भूल जाने के), बस बोलो:

```text
"ये सेव करो — vLLM सर्विंग ट्यूटोरियल: https://vllm.readthedocs.io/en/latest/
टैग्स: mlops, inference, vllm"

→ ✅ सेव हो गया: `links/2026-07-06_vllm-serving-tutorial.md`
   कैटेगरी: 🔗 लिंक्स
```

बाद में पूछो:

```text
"inference के मेमोस ढूंढो"

→ 🔍 मेमोबॉक्स में सर्च — 11 में से 2 मेमोस

  · 🔗 लिंक्स **vLLM सर्विंग ट्यूटोरियल**
    https://vllm.readthedocs.io/en/latest/
    📁 `links/2026-07-06_vllm-serving-tutorial.md`
```

**सीनारियो 3: "मेरे आइडियाज कितने हैं?"**

```text
"मेरे आइडियाज कितने हैं?"

→ 📊 आइडिया बॉक्स का स्टेटस

🥇 S ग्रेड: 0

  · 📋 [रीसेंट आइडिया 2]
  · 📰 [रीसेंट आइडिया 3]
  · …
```

### `search_ideas(keyword, limit=5)`

आइडिया बॉक्स की फाइल्स में टाइटल + कंटेंट से कीवर्ड सर्च करता है। टाइटल, समरी, ग्रेड (S/A/B/C), और फाइल पाथ रिटर्न करता है।

### `count_ideas()`

टोटल काउंट, ग्रेड डिस्ट्रीब्यूशन (S/A/B/C), और 5 सबसे रीसेंट आइडियाज रिटर्न करता है।

### `search_memos(keyword="", category="", limit=10)`

मेमो बॉक्स में सर्च करता है। ऑप्शनल `category` फिल्टर:
`links` · `news` · `quotes` · `references` · `snippets` · `specs` · `tips` · `misc`

### `count_memos()`

टोटल काउंट और पर-कैटेगरी ब्रेकडाउन रिटर्न करता है, साथ में 3 सबसे रीसेंट ऐडिशन्स।

### `add_memo(title, content, category="", tags="")`

नया मेमो ऐड करता है। अगर `category` खाली है, तो सर्वर ऑटो-डिटेक्ट करता है:
- URLs मौजूद हैं → `links`
- कोड/कमांड्स/कॉन्फिग्स → `snippets`
- सिटेशन्स / कोटेड टेक्स्ट → `quotes`
- न्यूज़ कीवर्ड्स → `news`
- स्पेक्स/रिक्वायरमेंट्स → `specs`
- हाउ-टू / एरर फिक्सेस → `tips`
- पेपर्स/बुक्स/लिंक्स → `references`
- डिफॉल्ट → `misc`

फाइल पाथ और डिटेक्टेड कैटेगरी रिटर्न करता है।

## 🚀 क्विक स्टार्ट

### इंस्टॉलेशन

```bash
pip install idea-box-mcp
```

या सोर्स से:

```bash
git clone https://github.com/nerin81-netizen/idea-box-mcp
cd idea-box-mcp
python3 -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
```

### पाथ्स कॉन्फ़िगर करो

डिफॉल्ट रूप से, सर्वर यहाँ से पढ़ता है:
- आइडिया-बॉक्स: `~/ideas-h/`
- मेमो-बॉक्स: `~/memos-h/`

कस्टमाइज़ करने के लिए, `server.py` के टॉप पर कॉन्स्टेंट्स एडिट करो:

```python
IDEA_DIR = Path("/तुम्हारा/पाथ/ideas-h")
MEMO_DIR = Path("/तुम्हारा/पाथ/memos-h")
```

### अपने MCP क्लाइंट में रजिस्टर करो

#### Claude Desktop

`~/Library/Application Support/Claude/claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "idea-box": {
      "command": "idea-box-mcp"
    }
  }
}
```

#### Cursor / Cline

`.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "idea-box": {
      "command": "idea-box-mcp"
    }
  }
}
```

## 🏗 आर्किटेक्चर

```
┌─────────────────┐      stdio/JSON-RPC      ┌──────────────────────┐
│  LLM Client     │ ◀──────────────────────▶ │  idea-box-mcp        │
│  (Claude/Cursor) │                          │  (FastMCP server)    │
└─────────────────┘                          └──────────┬───────────┘
                                                        │ pathlib
                                            ┌───────────┴────────────┐
                                            ▼                        ▼
                                    ┌──────────────┐         ┌──────────────┐
                                    │  ~/ideas-h/  │         │  ~/memos-h/  │
                                    │  *.md files  │         │  8 subdirs   │
                                    └──────────────┘         └──────────────┘
```

**डेज़ाइन चॉइसेस:**

- **stdio transport** — MCP स्टैंडर्ड
- **FastMCP** — Anthropic का ऑफिशियल Python SDK
- **MCP के अलावा ज़ीरो एक्सटर्नल डेप्स** — प्योर फाइल I/O
- **आइडिया-बॉक्स पर रीड-ओनली** (तुम्हारा स्ट्रक्चर प्रिज़र्व करता है) + **मेमो-बॉक्स पर राइट** (नई फाइल्स ऐड करता है)
- **ऑटो-कैटेगराइज़ेशन** सिंपल ह्यूरिस्टिक्स से (URLs / कोड / कीवर्ड्स)

## 🧪 डेवलपमेंट

```bash
# डेव डिपेंडेंसीज इंस्टॉल करो
pip install -e ".[dev]"

# stdio JSON-RPC को डायरेक्ट टेस्ट करो
echo '{"jsonrpc":"2.0","method":"tools/list","id":1}' | idea-box-mcp

# कैटेगराइज़ेशन लॉजिक का यूनिट टेस्ट

# → 'links'
print(_detect_memo_category('Bash script', 'pip install mcp'))
# → 'snippets'
"
```
